Git 팀플 branch 참고 명령어
브랜치 > 게임 pbe 서버 개념

현재 branch 위치 확인
	> git branch

branch 생성
	> git branch (새로운 브랜치명)

branch 이동
	> git checkout (이동할 브랜치명)

다른 branch의 내용을 합치기
★main에서★ (main에 합치고 싶다면)
	> git merge (병합할 브랜치명)

ㅁ	-	ㅁ	-	ㅁ	-	ㅁ
main		dev		받은dev		조원1 개인프랜치


main 브랜치 올리기
dev 브랜치 바로 올리기
조원은 dev 브랜치만 받기
	> git clone -b (들고 올 브렌치명) (원격저장소url)
조원과 조장 모두 dev 브랜치에서 나만의 브랜치 만들기
나만의 브랜치에서 작업이 끝난 후
dev 브랜치로 이동해서 나만의 브랜치와 합치기
@@@@@@@@@@@@@@@@@@@@@@@@
합친 dev 파일 보내기
다시 내 코드 작업 시 !!!!!!!!!!
내 컴퓨터 dev에서 변경된 dev 받기
나만의 브랜치에서 변경된 dev 병합하기
나만의 브랜치에서 작업하기
나만의 브랜치에서 작동이 잘 되는 지 확인하기
dev로 옮겨서 나만의 브랜치 병합하기
